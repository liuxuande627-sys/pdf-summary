# KWZero MVP产品需求文档 (精简版)

## 产品定位
**30秒获得低竞争关键词** - 无需AI，基于真实数据的极简关键词发现工具

## MVP核心参数
- **免费试用**: 3次关键词生成
- **付费方案**: $4.9/30次 (一次性付费，永久有效)
- **技术栈**: 无AI，规则引擎驱动
- **开发周期**: 5天完成

## 目标用户
- **主要用户**: 需要快速关键词建议的内容创作者
- **使用场景**: 博客写作、产品描述、SEO优化
- **付费意愿**: 愿意为节省时间付费的个人用户

   - 金融服务
   - 健康与保健
   - 科技产品
   - 教育培训

3. **计数系统**
   - 免费用户: 3次生成

## 技术架构 (无AI方案)

### 前端技术栈
- **HTML**: 纯HTML5语义化标签
- **CSS**: Tailwind CDN + 自定义苹果风格
- **JavaScript**: Vanilla JS零依赖
- **部署**: Vercel/Netlify一键部署

### 后端技术栈
- **Node.js**: Express轻量级服务
- **数据存储**: JSON文件存储词库
- **缓存**: 内存缓存 (无需Redis)
- **支付**: Stripe Checkout集成
- **计数**: 基于IP的简单计数器

### 数据源策略
```
预制词库构成:
├── 5000个预筛选低竞争关键词
├── 8个行业 × 600关键词/行业
├── 每个关键词包含: 词, 搜索量, 难度, CPC
├── 更新频率: 每周手动更新10分钟
└── 数据来源: Google官方免费工具导出
```

### 关键词算法
```javascript
// 无AI的评分算法
关键词评分 = (搜索量 × 商业价值) / (竞争度 + 1)
筛选条件: 难度<30 且 搜索量>100
排序方式: 评分降序
返回数量: 10个最佳关键词
```
